Document Object Model or DOM

Methods for target DOM (To GET and SET the value of html tags using js).(inside HTML file)

  • Most Important Target Methods
    • id : getElementById("")
    • Class Name : getElementsByClassName("")
    • Tag Name : getElementsByTagName("")
  • Basic Targit Methods
    • document.all : (return all tags in html document)
      • document.all[7] : target element in 7 index (according to its length)
    • document.images : (Same as document.all)
    • document.anchors : (Same as document.all)
    • document.links : (Same as document.all)
    • document.forms : (Same as document.all)
    • document.head : (return Head element)
    • document.title : (return title element)
    • document.body : (return body element)
    • document.URL : (Returns URL of wbsite)
    • document.baseURI : (Returns URL of wbsite)
    • document.domain : domain name like abcd.com
    • document.documentElement : return all document
    • document.doctype : (return document type like this doctype is html)
